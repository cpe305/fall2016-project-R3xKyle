Overview

Sudoku is a java application that generates random Sudoku puzzles in three different difficulties.

Features:

  • Easy game (4x4)
  • Medium game (6x6)
  • Hard game (9x9)
  • Best time high-score
  • Check game as you go
  • Solutions are randomly generated
